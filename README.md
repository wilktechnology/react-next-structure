## Overview

### Installation

```bash
$ npm install
$ yarn
```

### Running the app

```bash
# development
$ npm run dev
$ yarn dev

# production mode
$ npm run start
$ yarn start

# build project
$ npm run build
$ yarn build

# start json-server
$ npm run server
$ yarn server
```
